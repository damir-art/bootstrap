# Text
Классы для работы с текстом: https://getbootstrap.com/docs/5.2/utilities/text/

- text alignment,
- text wrapping and overflow,
- word break,
- text-transform,
- font-size,
- font-weight and font-style
- line-height
- monospace
- reset color
- text-decoration

## Text alignment

    text-start
    text-center
    text-end
    text-sm-start
    text-md-start
    text-lg-start
    text-xl-start

- `text-center text-md-start` - на маленьких экранах текст по центру, на больших слева.

## Reset color

    text-reset

# Папка Bootstrap
Что содержит папка Bootstrap?

Папка Bootstrap содержит две папки CSS и JS в которых лежат скомпилированные файлы `.css` и `.js`. Обычно из CSS используются файлы `bootstrap.css` или `bootstrap.min.css` а у JS `bootstrap.bundle.js` или `bootstrap.bundle.min.js`.

Обозначения:
CSS:
- `grid` - только сетка
- `rtl` - для текстов справа на лево, например арабских
- `rebot` - только сброс стилей
- `utilities` - только служебные CSS-классы

JS:
- `bundle` - включает Bootstrap.js и Popper.js

Общие:
- `min` - минимизированный файл
- `map` - файл исходной карты, помогает в работе с `инспектором` в браузере
